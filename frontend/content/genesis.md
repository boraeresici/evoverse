# Genesis starts as a seed, not a script.

Alpha begins from a deterministic seed: a finite region field, initial species pressure, resource distribution, and stability gradients. Evoverse does not spawn a hand-authored story. It starts a world state, then lets ticks, migration, mutation, collapse, recovery, and events turn that state into observable history.

## From Conway to Evoverse

Conway's Game of Life begins with a pattern of live and dead cells. Evoverse keeps the seed, grid, tick, and emergence spirit, but raises each map cell into an aggregate region. That region can carry energy, resources, stability, dominant species, population composition, and historical events.

This is why Genesis is not a binary pattern editor. It is the first ecological condition of Alpha.

## What the first seed contains

- A finite Alpha region field.
- Initial energy, resource, and stability differences.
- Early species populations distributed across origin regions.
- Traits that create pressure: efficiency, adaptation, cooperation, mobility, and resilience.
- Chronicle-ready events that explain what changed after the seed starts running.

## Manual or automatic Genesis

The current product direction is automatic Genesis for Alpha: users should arrive after the first seed has already become a living observatory. Manual Genesis can become an admin or future universe setup surface later, but it should not block the first experience.

## What users should feel

The first impression should be that Alpha is already alive. The landing page shows the current world. Universe shows the macro field. Region and species views let the user zoom into micro-life projections and history. Genesis explains why the world has this shape.
